## Description

Number Baseball Game is guessing player's number. You attempt to guess the other playerâ€™s number. \

### Validation check

Digits
- User should choose all different digits for the number.
ex) 1,2,2 or 4,1,4 (X)
- Name should be entered (can't be null)
